Initialization

__init__(self, token)

Initializes the UserApi instance with the provided token and sets up the headers for API requests.

Parameters:

  • token (str): Your Discord bot or user token for authentication.

Message Methods

sendMessage

sendMessage(self, message, channelID, replyID=None)

Sends a message to a specified channel. Optionally replies to a specific message.

Parameters:

  • message (str): The content of the message to be sent.
  • channelID (str): The ID of the channel where the message will be sent.
  • replyID (str, optional): The ID of the message to reply to (if any).

Returns:

Response object from the requests.post call.

getMessages

getMessages(self, channelID, limit=50)

Retrieves messages from a specified channel.

Parameters:

  • channelID (str): The ID of the channel from which messages will be retrieved.
  • limit (int, optional): The number of messages to retrieve (default is 50).

Returns:

A list of messages as SimpleNamespace objects.

Guild Methods

getGuilds

getGuilds(self)

Retrieves a list of guilds (servers) the user is part of.

Returns:

Response object from the requests.get call.

getGuildInfo

getGuildInfo(self, guildID)

Retrieves information about a specific guild.

Parameters:

  • guildID (str): The ID of the guild to retrieve information about.

Returns:

Guild information as a SimpleNamespace object.

Member Methods

kickMember

kickMember(self, guildID, userID)

Kicks a member from a specified guild.

Parameters:

  • guildID (str): The ID of the guild where the member is.
  • userID (str): The ID of the user to kick.

Returns:

Response object from the requests.delete call.

banMember

banMember(self, guildID, userID, reason=None)

Bans a member from a specified guild.

Parameters:

  • guildID (str): The ID of the guild where the member is.
  • userID (str): The ID of the user to ban.
  • reason (str, optional): The reason for the ban.

Returns:

Response object from the requests.put call.

Role Methods

createRole

createRole(self, guildID, name, permissions=0, color=0, hoist=False)

Creates a new role in a specified guild.

Parameters:

  • guildID (str): The ID of the guild where the role will be created.
  • name (str): The name of the role.
  • permissions (int, optional): The permissions assigned to the role (default is 0).
  • color (int, optional): The color of the role (default is 0).
  • hoist (bool, optional): Whether the role is displayed separately in the member list (default is False).

Returns:

Response object from the requests.post call.
